## 📌 Overview

**VitaeCraft** is a fully client-side resume builder built with pure HTML, CSS, and vanilla JavaScript. It helps job seekers create polished, ATS-friendly resumes in minutes — with real-time preview, an ATS score analyzer, AI-powered bullet improvements, and one-click PDF export.

Everything runs in the browser. Your data never leaves your device.

> Built for students, developers, designers, and professionals across every field — with demo templates pre-loaded for each career stage.

## ✨ Features

### 🎯 ATS Score Analyzer
Analyzes your resume across **8 weighted categories** and returns a score out of 100:

| Category | Max Points |
|---|---|
| Contact Information | 15 |
| Summary / Profile | 15 |
| Skills Section | 15 |
| Work Experience & Bullets | 20 |
| Quantified Metrics | 10 |
| Education | 10 |
| Certifications | 5 |
| Achievements | 5 |
| Format Safety | 5 |

Actionable feedback is shown for every category so you know exactly what to fix.

---

### ✨ AI Bullet Improver
Paste a weak bullet point and instantly get **3 stronger, metric-driven alternatives** powered by AI. Designed to turn vague responsibilities into compelling achievements.

---

### 🔍 JD Keyword Matcher
Paste any job description and get an instant list of **keywords missing from your resume**, so you can tailor your application to pass recruiter filters.

---

### 🎨 6 Professional Templates

| Template | Style |
|---|---|
| **Modern** | Bold header strip, clean sections, color-accented titles |
| **Classic** | Center-aligned header, traditional double-column layout |
| **Executive** | Dark sidebar, high-contrast, suited for leadership roles |
| **Minimal** | Ultra-clean, whitespace-first, single thin divider |
| **Compact** | Dense layout, optimized for one-page fits |
| **Creative** | Expressive layout for design and creative fields |

All templates are **ATS-safe** — no tables, no text boxes, no hidden characters.

---

### 💾 Auto-save + Named Snapshots
- Your resume auto-saves to `localStorage` on every keystroke
- Create **named snapshots** at any milestone ("Before Interview", "v2 with cover letter")
- Restore any saved version instantly — no data loss ever

---

### 📤 One-click PDF Export
Exports a **pixel-perfect, print-ready PDF** using the browser's native print engine. No watermarks. No sign-up. No upload.

## 📁 Project Structure

```
VitaeCraft/
│
├── index.html              # Landing page
├── app.html                # Resume editor app
│
├── css/
│   ├── landing.css         # Landing page styles
│   └── app.css             # Editor app styles
│
└── js/
    ├── landing.js          # Landing page interactions (scroll, animations)
    ├── app.js              # App bootstrap, toast, mobile toggle
    ├── state.js            # Global state object (resume data)
    ├── data.js             # Demo resume profiles (7 templates)
    ├── form.js             # Form inputs, event listeners, live binding
    ├── render.js           # Resume HTML renderer (all 6 templates)
    └── panels.js           # ATS panel, JD matcher, snapshot, AI improver
```

### Architecture Notes

- **No framework** — pure ES6 vanilla JavaScript, zero build tooling
- **State management** — single global `S` state object in `state.js`, mutated directly and re-rendered on change
- **Rendering** — `render.js` reads from state and produces resume HTML string, injected into the preview pane
- **Persistence** — `localStorage` keys `vitaecraft_v1_state` (live state) and `vitaecraft_v1_snaps` (snapshots)
- **Offline capable** — all assets are local or loaded from Google Fonts CDN

## 🛠 Getting Started

### Prerequisites
None. This is a zero-dependency project. You only need a modern browser.

### Run Locally

```bash
git clone https://github.com/nandandas2407-web/VitaeCraft.git
cd VitaeCraft
open index.html    # or just double-click the file
```

### Use as a Template
1. Fork this repository
2. Edit `index.html` and `app.html` to change branding
3. Modify `js/data.js` to change demo profiles
4. Add new templates in `js/render.js`
5. Deploy to GitHub Pages, Netlify, or Vercel with one click — no config needed

### Deploy to GitHub Pages

```bash
# In your forked repo settings:
# Settings → Pages → Source → Deploy from branch → main → / (root)
# Your site will be live at: https://<username>.github.io/VitaeCraft/
```

## 🔧 Customization

### Adding a New Resume Template

In `js/render.js`, add a new case to the template switch and return an HTML string:

```js
case 'mytemplate':
  return `
    <div class="resume-mytemplate">
      <h1>${data.name}</h1>
      <!-- your layout here -->
    </div>
  `;
```

Then register it in `app.html` as a new tab button with `data-tpl="mytemplate"`.

### Adding a New Demo Profile

In `js/data.js`, add a new entry to the `DEMOS` object:

```js
const DEMOS = {
  my_profile: {
    name: 'Jane Doe',
    title: 'Product Manager',
    email: 'jane@company.com',
    // ... rest of fields
  }
};
```
